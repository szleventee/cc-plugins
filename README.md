# cc-plugins

Claude Code plugin marketplace by levisz.

## Installation

```
/plugin add-marketplace https://github.com/levisz/cc-plugins
```

## Available Plugins

### expo-apps-buddy
Your friendly Expo app building buddy. Sets up projects with best practices and guides beginners through development with auto-saves, easy undo, and plain-language explanations.

**Skills included:**
- `expo-setup` — Scaffold a complete Expo project
- `beginner-guide` — Auto-save, undo, and beginner-friendly guidance

Install:
```
/plugin install expo-apps-buddy
```
