# expo-apps-buddy

Your friendly Expo app building buddy for Claude Code.

## Skills

### `expo-setup`
Sets up a complete Expo React Native project from scratch with Expo Router, modern UI best practices, web preview, and phone testing support.

### `beginner-guide`
Activates automatically when building Expo apps. Auto-saves progress after every change, helps undo mistakes, explains everything in plain language, and shows screenshots after every change.

## Getting Started

1. Create a new folder for your app
2. Run `/expo-apps-buddy:expo-setup myapp`
3. Start building — the beginner-guide activates automatically

## Requirements

- Node.js (v18+)
- Claude Code Desktop with Preview support
- Expo Go app on your phone (for mobile testing)
